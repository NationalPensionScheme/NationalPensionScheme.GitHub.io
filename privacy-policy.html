This site is a static website and does not directly collect any personally identifiable information (PII) from users. PII includes data that can be used to identify an individual, such as names, addresses, or email addresses.

However, please be aware of the following:

  • GitHub Pages: GitHub Pages, which hosts this site, may collect log data, including IP addresses and usage information, for operational purposes. Please refer to the GitHub Privacy Policy for more details.

  • Cookies: The static nature of this site, as currently deployed, does not utilize cookies. However, GitHub Pages or linked third-party services could potentially use cookies.
